Panduan lengkap pengelolaan platform KitaCatat dari sisi admin — user, transaksi, broadcast, dan monitoring.

01

Akses Admin Panel

Admin panel terpisah dari dashboard user biasa dan memerlukan kredensial khusus.

⚠️
Jangan bagikan kredensial admin ke siapapun. URL admin panel tidak dipublikasikan di halaman manapun untuk alasan keamanan.
1

Buka halaman login admin

Akses: kitacatat.masbendz.com/admin/login.php

2

Masukkan username & password admin

Kredensial dibuat saat setup awal via tools/admin_setup.php. Jika lupa password, reset langsung via phpMyAdmin di tabel admins.

3

Sesi admin terpisah

Login admin tidak mempengaruhi sesi user biasa. Bisa login sebagai admin dan user sekaligus di browser berbeda.

04

Manajemen Transaksi

Lihat semua transaksi platform dan pulihkan transaksi yang terhapus.

Cara Restore Transaksi Terhapus

1

Buka halaman Transaksi

Menu Admin → Transaksi

2

Pilih filter "Terhapus (Soft Delete)"

Di dropdown filter, pilih 🗑️ Terhapus (Soft Delete) lalu klik Cari

3

Klik tombol Pulihkan

Klik tombol ↑ Pulihkan di baris transaksi yang ingin dikembalikan. Transaksi akan aktif kembali dan muncul di dashboard user.

ℹ️
Soft delete berarti data tidak benar-benar dihapus dari database — hanya ditandai sebagai terhapus. Admin dapat memulihkan kapanpun selama data belum dibersihkan secara manual.

Pencarian Transaksi

Gunakan kolom pencarian untuk cari berdasarkan kode TXN, deskripsi, nama user, atau nomor WA. Filter aktif dan terhapus bisa dikombinasikan dengan pencarian.

05

Manajemen Kategori Global

Kelola kategori default yang tersedia untuk semua user.

⚠️
Hati-hati saat menghapus kategori. Jika kategori masih dipakai oleh transaksi, transaksi tersebut akan kehilangan kategorinya (menjadi kosong). Jumlah transaksi ditampilkan di setiap baris kategori.
AksiCara

Edit nama/iconKlik ikon pensil di baris kategori → ubah → Simpan
Hapus kategoriKlik ikon tempat sampah → konfirmasi
Format iconIsi dengan class Font Awesome tanpa prefix fa-, contoh: fa-car, fa-utensils
06

Broadcast WhatsApp

Kirim pesan ke semua user atau nomor tertentu melalui bot WA.

⏱️
Broadcast dikirim satu per satu dengan jeda 0.3 detik antar user untuk menghindari spam. Untuk 100 user, proses memakan waktu sekitar 30 detik. Halaman tidak boleh ditutup selama proses berlangsung.
TargetCara
Semua user aktifPilih "Semua User Aktif" di dropdown — jumlah user ditampilkan
Nomor tertentuPilih "Nomor WA Tertentu" → isi nomor satu per baris (format 628xxx)

Format Pesan WA

Gunakan format WhatsApp standar:

FormatHasil
*teks*Teks tebal
_teks_Teks miring
~teks~Teks dicoret
Baris baruEnter di textarea — dipertahankan di WA
07

Support Chat

Balas pesan dari user yang menghubungi via dashboard atau via WA dengan perintah admin: ...

1

Lihat pesan masuk

Menu Admin → Support Chat. Badge merah di menu menandakan ada pesan baru yang belum dibaca.

2

Pilih percakapan user

Klik nama user di panel kiri. Pesan ditampilkan seperti tampilan chat pada umumnya.

3

Balas pesan

Ketik balasan di kolom bawah → Enter atau Ctrl+Enter untuk kirim. Balasan otomatis dikirim ke WA user sekaligus tersimpan di riwayat chat.

📱
Tombol Buka WA di pojok kanan chat membuka WhatsApp langsung ke nomor user — berguna jika perlu komunikasi di luar platform.
08

Jika lupa password admin, reset manual via phpMyAdmin:

UPDATE `admins` SET `password` = '$2y$10$...' WHERE `username` = 'admin';

Hash password baru bisa dibuat dengan upload sementara file tools/hash.php berisi <?php echo password_hash('passwordbaru', PASSWORD_BCRYPT); ?>
